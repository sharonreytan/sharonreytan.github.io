The Hard Parts I Wrote Myself

Everything here was built with strong teams — these are the pieces I wrote with my own hands. (The work also produced 3 patent filings I co-invented; the safety gate below is mine from concept to code.)

The agent safety gate

Patent filed — concept to code

How do you let an AI agent act inside a live, authenticated banking session — without trusting it? My answer: the agent never touches the browser. Every proposed action is validated against an allowlist gate before a deterministic executor runs it; rejections feed precise feedback back to the agent. The gate is never skipped — and its rejection log kept surfacing legitimate new paths, so it grew coverage instead of just blocking.

The eval framework & observability

Measure before you steer. I built much of the evaluation framework that became the product's steering signal — per-provider ground truth, full-session scoring, repeat runs for consistency. It's what proved the first agents at ~98% precision/recall — and what justified killing the approach anyway on cost. Plus the observability tool: one correlation id → the full session audit, DOMs, screenshots, and highlighted agent actions.

The xpath generator

Screen-scraping lives or dies on selectors. I wrote the generator that crafts xpaths robust enough to survive layout changes yet precise enough to never grab the wrong element — sometimes embedding entity data into the selector itself to enforce correctness. The code-generation pipeline runs on it.

The Elasticsearch library & the sync engine

At AccessFintech I wrote the org-wide Elasticsearch library — a proprietary filter language so any engineer could query a famously unforgiving datastore safely — and rebuilt the sync engine joining-on-write across the entity graph from MongoDB into Elasticsearch. The parts I'm proudest of: a per-ID lock manager that cut lock waits 10 s → 0.5 s, a priority queue + latest-version-wins entity store that brought UI latency to ~2 s, and partial document updates in Elasticsearch — a mechanism that didn't exist; we invented it. It's the work that won me the infrastructure team.
